# instrument-nginx

sample based on ruby and docker compose where we add and test nginx instrumentation module(s) [HTTP, Websocket]


### Building

    docker-compose build
    
    
### Running

    docker-compose up 
    

then visit http://localhost


### Specs

    cd spec
    bundle exec rake
